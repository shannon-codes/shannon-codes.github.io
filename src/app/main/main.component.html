iOS Movie Night Mobile App

An iOS mobile app that helps users plan their movie night. It calls themoviedb.org API to display information about movies that are upcoming, now playing, and top rated. The user can view, save, delete their list of favorites. They can sets push notification reminders on movie release dates. There is a map feature to find directions. For this project I worked in team of 4.

Built with Swift, SQLite.

Student Grades Management Portal

Created web app for class management for students and a professor. It has the ability for students to view their grades. As for professors, they can view, add, edit, and delete new and existing students and grades. They can see the class GPA and average grades per assignment. It has login authentication and user role authorization. stores encrypted password, uses Thymeleaf template engine to assist with form binding to java bean, and Lombok to simplify java bean by redusing boilerplate code

Built with Java, Spring Boot, Hibernate (ORM), H2 In-Memory database and on for frontend using HTML, CSS, Bootstrap

Health Diary

This application allows users to complete a daily questionnaire entry on health-related metrics such as glasses of water, meals, hours of sleep, calories burned, amount of exercise, etc. It allows the user to create, read, update, delete, and search for diary entries. Knight's Tour Simulator

This application runs simulations of the Knight's Tour problem. The aim is the move the knight to every position on the chessboard, but the knight can visit a square only once. It allows the user to choose the simulation method and the knight's starting position. The result displays the sequence of movements of the knight on the chessboard and the total number of squares visited. The results are saved into a text file.

Built with C# .NET, Windows Forms
